# Quad Solutions – Medical Credentialing Website

A complete, multi-page corporate website for a medical credentialing company. Designed to help healthcare providers get credentialed with insurance networks faster.

🔗 **Live Demo:** [https://quadsolution1.vercel.app/](https://quadsolution1.vercel.app/)  
📦 **GitHub Repository:** [https://github.com/Ayesha-Amjad-123/quadsolution](https://github.com/Ayesha-Amjad-123/quadsolution)

---

## 📌 Project Overview

Quad Solutions is a **fully responsive static website** built with HTML, Tailwind CSS, and vanilla JavaScript. It features separate pages for all core services, a 3D interactive globe, animated counters, a stack slider, lead generation forms, and a consistent professional design.

The site is deployed on **Vercel** and can be easily customised or extended.

## ✨ Key Features

### Interactive Elements
- **3D Rotating Globe** – Medical icons arranged on a sphere that rotates with mouse movement (homepage only)
- **Stack Slider** – Horizontal service preview panel with auto‑rotate and hover pause
- **GSAP Scroll Animations** – Fade-up, fade-left, fade-right, and zoom‑in effects as you scroll
- **Animated Counters** – Statistics (providers credentialed, success rate, etc.) count up when they enter the viewport

### Trust & Credibility
- Certification badges (NCCB Certified, HIPAA Compliant, NCQA Standards, A+ Rated)
- Client testimonials with star ratings
- Stats section with real numbers
- Trust badges (98% success rate, 30‑day average timeline)

### Lead Generation
- **Lead forms on every page** – Consultation request forms throughout the site
- Form submission shows a success message (ready for backend integration)

### Design & UX
- Fully responsive (desktop, tablet, mobile)
- Accessibility‑friendly (focus outlines, semantic HTML)
- Consistent header, footer, and colour scheme (Teal/White/Gray)
- Professional healthcare aesthetic
